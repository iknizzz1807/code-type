# CodeType

Practice typing real code snippets tailored to your programming language and work.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file with your Gemini API key:
   ```bash
   cp .env.example .env
   # Edit .env and add your API key
   ```

3. Run the dev server:
   ```bash
   npm run dev
   ```

## Features

- 12 programming languages supported (Python, Rust, TypeScript, Go, Java, C++, C#, Ruby, PHP, Swift, Kotlin, Zig)
- AI-generated code snippets tailored to your profile
- Syntax highlighting
- WPM and accuracy tracking
- Progress history
